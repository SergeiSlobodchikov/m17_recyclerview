MainFragment: Этот фрагмент отвечает за отображение списка фотографий Марса. Он использует PhotoAdapter для отображения каждой фотографии и MainViewModel для получения данных о фотографиях от Repository.

MainViewModel: Этот класс ViewModel используется для управления данными и состоянием приложения. Он взаимодействует с Repository для получения списка фотографий и обработки ошибок.

Repository: Этот класс используется для взаимодействия с внешним API, который предоставляет данные о фотографиях Марса. Он использует Retrofit и Moshi для преобразования JSON-ответов в объекты Kotlin.

PhotoAdapter: Этот адаптер используется для отображения списка фотографий в RecyclerView. Он также обрабатывает нажатия на элементы списка.

PhotoFragment: Этот фрагмент отображает подробную информацию о выбранной фотографии. Он принимает аргументы из навигации и использует их для отображения информации о фотографии.

MainActivity: Это основной класс активности, который запускает приложение. В данном случае он просто устанавливает содержимое главного экрана.
